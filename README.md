# Introduction

There are many, many song and lyric repositories online that are freely
available to browse. However, many of them are hard to use. The data is
availalbe, but it is not useful. This project aims to create a set of tools that
can be applied to these sources of data.

I tried an anchoring approach to parsing the data, written mostly by me and some
ChatGPT, and it mostly failed. ChatGPT suggested a scoring method to guess what
lines are lyrics or not.
