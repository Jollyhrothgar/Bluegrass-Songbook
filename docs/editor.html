Format Help
  • [G] - Place chord before the syllable
  • [Am7] - Any chord notation works
  • [G/B] - Slash chords supported
  • Blank line = new verse/section
  • {sov} / {eov} - Explicit verse markers
  • {soc} / {eoc} - Chorus markers
