OTF Tablature Editor

A modal, keyboard-accelerated editor for bluegrass tablature

Keyboard Shortcuts

  • h j k l or arrow keys - Navigate
  • i - Enter INSERT mode
  • Escape - Return to NORMAL mode
  • 1-5 - Select string (in INSERT mode)
  • 0-9 - Enter fret number (in INSERT mode)
  • Space - Advance cursor (in INSERT mode)
  • r - Enter ROLL mode for quick pattern entry
  • T I M R - Thumb/Index/Middle/Ring (in ROLL mode)
  • x - Delete note
  • u - Undo
  • q e s - Set duration (quarter/eighth/sixteenth)
  • ? - Show help

Double-click anywhere on the tablature to open the note entry popover.
